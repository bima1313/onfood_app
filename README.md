# onfood

A Food Ordering app project. This project using dependencies: 
* firebase_core,
* cloud_firestore,
* provider,
* etc

The minor update are:
* adding images on `Detail Order View`
* Performance optimization
* Splitting the code into the pieces
* Changing the splash screen background color into dark brown
